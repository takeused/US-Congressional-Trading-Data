# 컨텍스트 노트 — 미 의회 매매 데이터 파이프라인

## 프로젝트 개요
미국 하원 사무처 재무공시(PTR)를 공식 배포 파일에서 다운로드·파싱해 구조화 JSON을 만들고, 인리치·대시보드까지 만드는 파이프라인. 근거 문서: `미 의회 매매 데이터 — 공식 소스 파싱 가이드.md`.

## 확정 결정 (2026-07-19)
- **범위** — 파서(①~④) + 인리치(⑤) + HTML 대시보드 전체.
- **연도 처리** — 특정 연도 고정 없이 `--year` CLI 인자로 받아 재사용 가능하게.
- **작업 문서** — checklist.md, context-notes.md 먼저 생성 후 코딩.

## 환경 (확인 완료)
- Python 3.12.10, pdftotext 4.06 (poppler) 설치됨.
- yfinance 미설치 → ⑤ 단계 전 `pip install` 필요.
- tesseract / pytesseract 미설치 → 스캔 PDF OCR fallback은 보류. 우선 스캔 PDF는 플래그만 남기고 스킵, OCR 필요성 확인되면 도입.

## 아키텍처 결정
- 단계별 모듈 분리 (`src/fetch_index.py`, `fetch_pdf.py`, `extract.py`, `parse.py`, `enrich.py`, `pipeline.py`) — 각 단계 독립 검증·재실행 가능.
- 표준 라이브러리 우선 (urllib, zipfile, xml.etree, re, subprocess). 외부 의존은 yfinance만.
- PDF 로컬 캐시로 재파싱 시 재다운로드 방지 (서버 매너 + 속도).

## 구현 중 발견 (2026-07-19)
- **실제 PDF 형식이 가이드 예시보다 복잡** — 거래유형에 `S (partial)` 수식어, 소유자 열 `SP`/`JT`/`DC`, 티커·자산코드가 다음 줄로 줄바꿈. TXN_RE에 `(?:\([^)]*\)\s+)?` 추가로 해결.
- **티커 두 형식** — 괄호형 `(BRK.B) [ST]` + 거래소접두형 `NYSEARCA: DIA [OT]`. 둘 다 지원.
- **티커 오염 방지** — 설명 블록(F/S/O/D:)·빈 줄 전까지만 창을 확장. 서술에 나열된 다른 티커가 딸려오는 것 차단.
- **주식 티커 추출률 100%** (30건 샘플 검증). null 티커는 국채·회사채·사모펀드 등 본래 티커 없는 자산 → 정상.
- **자산코드 맵 확장** — 가이드의 ST/OP/MF/GS 외 실측된 OT/CS/HN/PE/EF/RP 추가 (config.py).
- **수익률 부호** — 매도는 반전(판 뒤 하락하면 이득). enrich.py에서 처리.
- **스캔 PDF** — 이번 2026 배치엔 없었음(OCR 미도입 유지).

## 스캔 PDF OCR 도입 (2026-07-19)
- **스캔 33개 = 전부 구형 종이 체크박스 양식** (DocID 8xxx·9xxx). 전자 PTR과 형식 완전히 다름.
- **엔진 선택** — PyMuPDF(렌더) + RapidOCR(ONNX, pip·오프라인). tesseract/poppler-img 미설치라 시스템 바이너리 없는 경로 채택. 사용자 승인.
- **복원 범위 = Tier1** (사용자 결정): 자산명·티커·거래일·공시일·소유자. 거래유형·금액은 체크박스 X라 미복원(null).
- **중대 발견 — asset_name 누락** — 처음엔 OCR 87% "중복"으로 보였으나, 실제론 자산명을 저장 안 해서 서로 다른 거래가 같은 튜플로 뭉친 착시. 41페이지 고빈도 트레이더 리포트라 행수(3768)는 대체로 실제. → Transaction에 `asset_name` 추가(전자·스캔 공통).
- **스캔 티커는 괄호형만** — 트레일링 심볼 휴리스틱이 Common→CMN, US Treasury→US 노이즈 대량 생성. "안정적" 우선이라 정밀도 택함.
- **소스 구분** — `source` 필드(text/ocr). OCR 행은 asset_type=None·type=""이라 대시보드 주식/수익률/섹터 통계에서 자연 제외. 대시보드 요약에 "OCR 복원" 카운트만 표기.
- **속도** — OCR 33개(일부 41p) ~10분+. `--ocr` 옵션으로 기본 실행과 분리.

## 통계 정직성 원칙 (가이드 §7)
- 모든 수익률에 표본수 n 병기, 소표본(n<5) 상위 노출 제외.
- "과거 성과가 미래 보장 아님" 고지. 정보 제공용, 투자 권유 아님.
